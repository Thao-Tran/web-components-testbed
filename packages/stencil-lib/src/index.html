Theme

test

A theme can be applied to the headless-button by using it inside a theme-provider element.

test

This will allow the priority attributes to actually affect the styling of the buttons.

The theme will not be applied to overridden root slot children.

Theme - properties property

The priority field set inside the properties property will be applied as well.

Theme - Composable themes

test

The theme-provider provides a button slot that allows for just the button theme to be overridden.

Styles from the theme-provider are scoped to the specific instance's children, so the theme-provider can be used multiple times in the same page with different themes.
